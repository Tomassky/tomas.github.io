032：Kali Linux操作系统识别

发表于 2019-12-06 | 分类于 Kali

  1. 操作系统识别

    • 操作系统识别技术（默认开放的系统开放端口）
      • 种类繁多
      • 好产品采用多种技术组合
    • TTL起始值
      • Windows：128（65-128）
      • Linux/Unix：64（1-64）
      • 某些Unix：255
TTL值劫持路由器，使不同地区的人访问到网页不一样

  1. 操作系统识别

    • Python
      • form scapy.all import*
      • win=”IP地址”
      • linx=”IP地址”
      • aw=sr1(IP(dst=win)/ICMP())
      • al=sr1(IP(dst=linu)/ICMP())
      • if al[IP].ttl<=64:
      • print “host is linux”
      • else:
      • print “host is windows”
    • ./ttl_os.py IP地址

  1. 操作系统识别

    • nmap使用多种技术识别操作系统
      • 🔺命令：nmap IP地址 -O
      • 系统服务特征
CPE国际标准化组织进行的标准化操作系统，设备类型分类

  1. 操作系统识别

    • 🔺命令：xprobe2 IP地址
    • 结果有误差

  2. 操作系统识别

    • 被动操作系统识别
      • IDS
      • 抓包分析
    • 被动扫描（不主动向目标主机发包，部署在网络出口的地方）
    • 🔺命令：p0f
      • 结合ARP地址欺骗识别全网OS

  3. SNMP扫描（通信：101和102，被查询的服务端：161，客户端：162 → udp）

    • snmp（明文传输，可被嗅探到）
      • 信息的金矿（设备的详细内容）
      • 经常被错误配置（特征字符，community）
      • public/private/mnager
    • MIB Tree
      • SNMP Management Information Base（MIB）
      • 树形的网络设备管理功能数据库
      • 1.3.6.1.4.1.77.1.2.25
    • 🔺命令：onesixtyone IP地址 public
    • 🔺命令：onesixtyone -c（字典） dict.txt -i（多个主机列表） host -o my.log -w 100
Window XP安装 → 程序软件安装 → 添加组件 → 管理和监视（有可能需要安装光盘）
services.msc snmp服务，可添加代理，陷阱（snmp作为客户端），默认安装public作为commuity，然后接受来自任何主机的SNMP查询包

  1. SNMP扫描

    • 🔺命令：snmpwalk IP地址 -c public -v（snmp版本） 2c
    • 用户
      • 🔺命令：snmp -c publuc -v 2c IP地址 1.3.6.1.4.1.77.1.2.25（MIB All ID）
    • 🔺命令：snmpcheck -t（目标主机） IP地址（默认情况下是public，可以用-c去修改community）
    • 🔺命令：snmpcheck -t IP地址 -c private -v 2
    • 🔺命令：snmpcheck -t IP地址 -p agent w（检测是否有可写权限）
SNMP版本：v1，v2c，v3（大量安全机制）
