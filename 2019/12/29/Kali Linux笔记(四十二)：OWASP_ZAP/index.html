Kali Linux笔记(四十二)：OWASP_ZAP

发表于 2019-12-29 | 分类于 Kali
字数统计: 715 字 | 阅读时长 ≈ 2 分钟

1. OWASP_ZAP

  • Zed attack proxy
  • WEB Application集成渗透测试和漏洞挖掘工具
  • 开源免费跨平台简单易用
  • 截断代理（自动配置端口）
  • 主动（Quick Start/站点目录 → 攻击 → Active Scan）
  • 被动扫描（手动爬网，截断代理）
  • Fuzzy（工具 → Fuzz/Request → 选择 → 右键 → Fuzz）▲ 注意File Fuzzers
  • 暴力破解（站点目录 → Request → 选择 → 右键 → Fuzz → Payload(字典文件)）
  • API（必须使用了OWASP_ZAP代理的情况下）
    • http://zap/

2. OWASP_ZAP

  • Persist Session
  • Mode
    • Safe（安全扫描，不包含危险扫描）
    • Protected（指定Web Application进行危险扫描）
    • Standard（标准扫描，存在一些危险扫描，可能造成破坏）
    • ATTACK（能扫就扫，能有多强烈就多强烈）
  • 升级Add-ones
  • Anti CSRF Tokens（每次访问，随机生成一个Token → 防止CSRF/重放攻击）
▲ 一般情况下，虽然Tokens是变化的，但Tokens的头部是一般是固定不变的，可以添加到OWASP_ZAP中，可以做到重放攻击
设置Options → Anti CSRF Tokens → Add（Tokens的名字）
  • https–CA（增加OWASP_ZAP自签名证书）
▲ 设置Options → Dynamic SSL Certificates → 保存（cet证书） → 安装证书
  • Scope/Cpntexts/Filter
  • Http Sessions–default session tokens & site session tokes
  • Note/tag
  • Passive Scan

3. OWASP_ZAP Session

  • persist the ZAP Session（文件 → 打开会话）
    • current timestamp（保存默认的ZAP目录）
    • specify the name and location（长久保存自定义目录）
    • not persist the ZAP Session（不长久保存）
  • Request/Response
  • 手动爬网 → OWASP_ZAP分析

4. OWASP_ZAP Manage Add-ones

  • Installed（版本更新可能需要到Github）
    • Uninstall Selected
    • Update Selected
  • Marketplace
    • Release（长期使用，正式发布）
    • Beta（测试使用，较正式发布）
    • Alpha（不是成熟的版本，稳定性准备性不是很高）

5. OWASP_ZAP policy

  • General
    • 添加策略
    • Name
    • Default Alert Threshold（报警阈值，一般选中等）
    • Default Attack Strength（攻击强度）
  • 信息收集
  • 客户端浏览器
  • 服务器安全
  • 杂项
  • 注入

6. OWASP_ZAP Scope/Cpntexts/Filter（范围/上下文/筛选）

  • 站点目录 → Include in Context → New context
  • Scope包含Context，可以删除一个Context
  • 同一个范围Scope同一个策略

7. OWASP_ZAP Http Session

  • 工具栏 → Session Properties
    • 针对不同Context有不同的Authentication
    • 默认Manual Authentication（记录访问过网站的Session，手动输入记录）
    • 支持表单、HTTP Basic、Sript-Base
  • 站点 → 加号 → 新建脚本
  • 选项 → Http Session → 增加Web Application自定义的Session

8. OWASP_ZAP 编码解码哈希

  • 选择编码的字段或要编码的字段 → 右键选择
SHA1 -- 彩虹表，统计哈希值对应的值

9. OWASP_ZAP 截断

  • 工具栏 → Set break on all requests and responses
  • 单步截断/一次截断
  • ▲ 显示隐藏域（小灯泡）– 代理服务器修改返回的内容后返回给浏览器

10. OWASP_ZAP

  • 标准扫描工作流程
    • 设置代理
    • 手动爬网
    • 自动爬网（OWASP_ZAP自动爬网，站点目录 → 右键攻击 → 爬网/Forced Browser site/Forced Browser directory/Forced Browser directory(and children)）
    • 主动扫描（OWASP_ZAP自动爬网，站点目录 → 右键攻击 → Active Scan）
