1. 发现-三层发现（ping）

  • 优点
    • 可路由
    • 速度比较快
  • 缺点
    • 速度比二层慢
    • 经常被边界防火墙过滤（目标系统的过滤机制）
  • IP、ICMP协议
ICMP协议，网络控制管理协议，路径发现，探测，根据相应结果来反应目标主机的状态
ICMP协议，发送端的报文类型是询问报文，Type字段为8，接收端为0
网络上默认禁止ICMP的探测数据包，所以导致扫描结果不准确，存在误报，漏报，错报
  • 🔺命令：Ping IP地址 -c 2
  • 🔺命令：Ping -R（路由追踪） IP地址 / traceroute IP地址
  • 🔺命令：ping IP地址 -c 1 | grep "bytes from" | awk '{print $4}'
  • 脚本
    • 🔺命令：./pinger.sh IP网段 ▲2019.12.2 iplist文档中ip地址后面有：，还无法去除
#!/bin/bash
if [ "$#" -ne 1 ];then
    echo "Usage - ./pinger.sh [/24 network address]"
    echo "Example - ./pinger.sh 192.168.100.0"
    echo "Example will perform an ICMP ping scan of the 192.168.100.0/24 network"
    exit
fi
prefix=$(echo $1 | cut -d "." -f 1-3)
for add in $(seq 1 254);do
  ping -c 1 $prefix.$addr | grep "bytes from" | awk '{print $4}'  &
done
Linux/Unix数据包的TTL值默认为64
ping -R 请求离主机比较远的网卡的IP地址（同个路由器）
traceroute 请求离主机比较近的网卡的IP地址（同个路由器）

2. 发现-三层发现（Scapy）

  • Scapy
    • OSI多层堆叠手工声称ICMP包-IP/ICMP包
    • ip=IP()
    • icmp=ICMP()
    • ping=(ip/icmp)
    • ping[IP].dst=”IP地址”
    • a=sr1(ping)
    • a.dispaly
    • Ping不存在的地址：a=sr1(ip/icmp,timeout=1)
    • a=sr1(IP(dst=”IP地址”)/ICMP(),timeout=1) //一句话实现
  • 🔺命令：./pinger1.py IP网段>iplist.txt
#!/usr/bin/python
import logging
import subprocess
logging.getLogger("scapy.runtime").setLevel(logging.ERROR)
from scapy.all import *

if len(sys.argv)!=2:
    print "Usage - ./pinger1.py [/24 network address]"
    print "Example - ./pinger1.py 192.168.100.0"
    print "Example will perform an ICMP ping scan of the 192.168.100.0/24 network"
    sys.exit()

address=str(sys.argv[1])
prefix=address.split(".")[0]+"."+address.split(".")[1]+"."+address.split(".")[2]+"."

for addr in range(1,254):
     a=sr1(IP(dst=prefix+str(addr))/ICMP(),timeout=1,verbose=0)
     if a==None:
       pass
     else:
       print prefix+str(addr)
  • 🔺命令：./pinger2.py iplist.txt
#!/usr/bin/python
import logging
import subprocess
logging.getLogger("scapy.runtime").setLevel(logging.ERROR)
from scapy.all import *

if len(sys.argv)!=2:
    print "Usage - ./pinger2.py [iplist.txt]"
    print "Example - ./pinger2.py iplist.txt"
    print "Example will perform an ICMP ping scan of the 192.168.100.0/24 network"
    sys.exit()

filename=str(sys.argv[1])
file=open(filename,"r")

for addr in file:
    a=sr1(IP(dst=addr.strip())/ICMP(),timeout=0.1,verbose=0)
    if a==None:
      pass
    else:
      print addr.strip()

5. 发现-三层发现（Hping）

  • Hping（几乎可以完成所以IP、TCP协议包头的组合，也可对于基于TCP服务器进行压力测试）
    • 能够发生几乎所有任意TCP/IP包
    • 功能强大但每次只能扫描一个目标
  • 🔺命令：hping3 IP地址 --icmp -c 2
  • 🔺命令：for addr in $(seq 1 254); do hping3 1.1.1.$addr --icmp -c 1 >> handle.txt & done
  • 🔺命令：cat handle.txt | grep ^len
