1. 端口扫描（TCP）

  • TCP端口扫描（SYN–SYN/ACK–ACK）

    • 基于连接的协议
    • 三次握手（ACK包连接）
    • 隐蔽扫描（隐藏与目标进程的会话）–SYN（开放 →SYN/ACK，不开放 → RST）
    1. 不建立完整连接（SYN（开放 →SYN/ACK，不开放 → RST））
    2. 应用日志不记录扫描行为–隐蔽（但审计网络层，还是可以发现）
    • 僵尸扫描（几乎不和目标进程尝试会话）
    1. 极度隐蔽
    2. 实施条件苛刻
    3. 可伪造源地址（▲扫描者与目标主机必须可以实现地址伪造）
    4. 选择僵尸机（▲必须要有，网络中闲置的机子，不同于僵尸网络的机子）
      • 闲置系统
      • 系统使用递增的IPID：（1）不能为0 （2）不能为随机
    • 全连接扫描（三次握手连接）
    • 所有的TCP扫描方式都是基于三次握手的变化来判断目标端口状态
IPID:IP报头里面的Identification字段
1.僵尸扫描过程（端口开放）：
（1）Scanner【SYN/ACK】 -- Zombie【RST IPID=X】
（2）Scanner【SYN IPSRC=zombie】 -- target【SYN/ACK】 -- Zombie【RST IPID=x+1,(僵尸机需要足够空闲)】
（3）Scanner【SYN/ACK】 -- Zombie【RST IPID=x+2】
2.僵尸扫描进程（端口未开放）：
（1）Scanner【SYN/ACK】 -- Zombie【RST IPID=X】
（2）Scanner【SYN IPSRC=zombie】 -- target【RST】 -- Zombie
（3）Scanner【SYN/ACK】 -- Zombie【RST IPID=x+1】

2. 隐蔽端口扫描

  • syn–syn/ack–rst（本机操作系统内核产生的）
  • Scapy
    • sr1(IP(dst=”IP地址”)/TCP(dport=80,flags=”S”),timeout=1,verbose=0)
    • 🔺命令：./syn_scan.py 目标IP地址 起始端口 结束端口
#!/usr/bin/python
import logging
logging.getLogger("scapy.runtime").setLevel(logging.ERROR)
from scapy.all import *
import time
import sys

if len(sys.argv)!=4:
    print "Usage - ./syn_scan.py [Target IP] [First IP] [Last IP] "
    print "Example - ./syn_scan.py 192.168.100.1 1 200"
    print "Example will perform a TCP SYN scan of the 192.168.100.0/24 range"
    sys.exit()

ip=sys.argv[1]
start=int(sys.argv[2])
end=int(sys.argv[3])

for port in range(start,end):
    a=sr1(IP(dst=ip)/TCP(dport=port),timeout=1,verbose=0)
    if a==None:
      pass
    else:
      if int(a[TCP].flags)==18:
        print port
      else:
        pass
正常情况下：
端口开放：syn--syn/ack
端口未开放：syn--rst/ack

5. 全连接端口扫描

  • Scapy（容易触发目标主机的报警系统）
    • syn扫描不需要raw packets
    • 内核认为syn/ack是非法包，直接发rst终断连接
    • 全连接扫描对scapy比较困难
  • sr1(IP(dst=”IP地址”)/TCP(dport=22,flags=”S”))
  • 🔺命令：./tcp_scan1.py（代码内指定IP地址）
#!/usr/bin/python
import logging
logging.getLogger("scapy,runtime").setLevel(logging.ERROR)
from scapy.all import*

response=sr1(IP(dst="IP地址")/TCP(dport=80,flags="S"))
reply=sr1(IP(dst="IP地址")/TCP(dport=80,flags="A",ack=(response[TCP].seq+1)))
  • 🔺命令：./tcp_scan2.py（代码内指定IP地址）
#!/usr/bin/python
import logging
logging.getLogger("scapy,runtime").setLevel(logging.ERROR)
from scapy.all import*

SYN=IP(dst="IP地址")/TCP(dport=25,flags="S")

print "--SENT--"
SYN.display()

print "\n\n--RECEIVED--"
response=sr1(SYN,timeout=1,verbose=0)
response.display()

if int(response[TCP].flags)==18:
    print "\n\n--SENT--"
    A=IP(dst="IP地址")/TCP(dport=25,flags="A",ack=(response[TCP].seq+1))
    A.display()
    print "\n\n--RECEIVED--"
    response2=sr1(A,timeout=1,verbose=0)
    response2.display()
else:
    print "SYN-ACK not returned"
  • 🔺命令：iptables -A OUTPUT -p tcp --tcp-flags RST RST -d 目标IP地址 -j DROP
iptables是工作在底层的防火墙，在内核之前过滤数据
iptables -L查看当前防火墙规则

9. 僵尸扫描

早期的Window系统，包括XP,2003,98系统的IPID是递增的
  • Scapy–zomebie.py
    • i=IP()
    • t=TCP()
    • rz=(i/t)
    • rt=(i/t)
    • rz[IP].dst=”zombie IP地址”
    • rz[TCP].dport=445
    • rz[TCP].flags=”SA”
    • rt[IP].src=”zombie IP地址”
    • rt[IP].dst=”target IP地址”
    • rt[TCP].dport=22
    • rt[TCP].flags=”S”
    • az1=sr1(rz) / at=sr1(rt,timeout=1) / az2=sr1(rz)
    • az1.display() / az2.display()
  • 🔺命令：./zombie.py
#!/usr/bin/python
import logging
logging.getLogger("scapy,runtime").setLevel(logging.ERROR)
from scapy.all import*

def ipid(zombie):
    reply1=sr1(IP(dst=zombie)/TCP(flags="SA"),timeout=2,verbose=0)
    send(IP(dst=zombie)/TCP(flags="SA"),verbose=0)
    reply2=sr1(IP(dst=zombie)/TCP(flags="SA"),timeout=2,verbose=0)
    if reply2[IP].id==(reply1[IP].id+2):
      print "IPID sequence is incremental and target appears to be idle.ZOMBIE LOCATED"
      response=raw_input("DO you want to use this zombie to perfore scan?(Y or N):")
      if response=="Y":
        target=raw_input("Enter the IP Address of the target system:")
        zombiescant(target,zombie)
    else:
      print "Either the IPID sequence is not incremental or the target is not idle.NOT A GOOD ZOMBIE"

def zombiescan(target,zombie):
    print "\nScanning target "+target+" with zombie "+zombie
    print "\n-----------Open Ports On Target-------------\n "
    for port in range(1,100):
      try:
        start_val=sr1(IP(dst=zombie)/TCP(flags="SA",dport=port),timeout=2,verbose=0)
        send(IP(src=zombie,dst=target)/TCP(flags="S",dport=port),verbose=0)
        end_val=sr1(IP(dst=zombie)/TCP(flags="SA",dport=port),timeout=2,verbose=0)
        if end_val[IP].id==(start_val[IP].id+2):
          print port
      except:
        pass

print "----------------Zombie Scan Suite-----------\n"
print "1 - Identify Zombie Host\n"
print "2 - Perform Zombie Scan\n"
ans=raw_input("Select an Option (1 or 2):")
if ans=="1":
    zombie=raw_input("Enter IP Address to test IPID sequence")
    ipid(zombie)
else:
    if ans=="2":
      zombie=raw_input("Enter IP Address for zombie sysytem:")
      target=raw_input("Enter IP Address for scan target:")
      zombiescan(target,zombie)
sr1 发包只收一个数据包
send 发包不收数据包
