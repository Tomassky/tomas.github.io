1. 端口扫描（UDP）

  • 端口对应网络服务及应用端程序
  • 服务端程序的漏洞通过端口攻入
  • 发现开放的端口
  • 更具体的攻击面
  • UDP端口
    • 假设ICMP port-unreachable 响应代表端口关闭（没有响应，即为端口开放）
    • 目标系统不响应ICMP port-unreachable时，可能产生误判
  • 完整的UDP应用层请求
    • 准确性高
    • 耗时巨大

2. 端口扫描（Scapy-UDP）

  • Scapy UDP Scan
    • 端口关闭：ICMP port-unreachable
    • 端口开放：没有回包
    • 了解每一种基于UDP的应用包结构很有帮助
    • 与三层相同的技术
    • 误判
  • Scapy
    • sr1(IP(dst=”IP地址”)/UDP(dport=53),timeout=1,verbose=1)
  • 🔺命令：./udp_scan.py IP地址 起始端口 结束端口
#!/usr/bin/python
import logging
logging.getLogger("scapy.runtime").setLevel(logging.ERROR)
from scapy.all import *
import time
import sys

if len(sys.argv)!=4:
    print "Usage - ./udp_scan.py [Target IP] [First IP] [Last IP] "
    print "Example - ./udp_scan.py 192.168.100.1 1 200"
    print "Example will perform a TCP ACK scan of the 192.168.100.0/24 range"
    sys.exit()

ip=sys.argv[1]
start=int(sys.argv[2])
end=int(sys.argv[3])

for port in range(start end):
    a=sr1(IP(dst=ip)/UDP(dport=port),timeout=5,verbose=0)
    time.sleep(1)
    if a==None:
      print port
    else:
      pass

3. 端口扫描（Nmap-UDP）

  • Nmap
  • 🔺命令：nmap -sU（端口扫描） IP地址
    • 默认的1000个参数（不指定端口的情况下）
    • ICMP host-unreachable
  • 🔺命令：nmap IP地址 -sU -p（指定端口） 53
  • 🔺命令：nmap -iL iplist.txt -sU -p（指定端口范围） 1-200
