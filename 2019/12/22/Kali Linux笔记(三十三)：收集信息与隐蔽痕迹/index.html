Kali Linux笔记(三十三)：收集信息与隐蔽痕迹

发表于 2019-12-22 | 分类于 Kali
字数统计: 800 字 | 阅读时长 ≈ 3 分钟

1. 信息收集

  • 应用系统的配置文件
    • 应用连接数据库的配置文件
    • 后台服务运行账户

2. 基本信息收集

  • Linux
    • /etc/resolv.conf（DNS信息）
    • /etc/passwd（存了用户账号和密码）
    • /etc/shadow（存放的密码信息）
    • 🔺命令：whoami,who -a
    • 🔺命令：ifconfig -a，iptables -L -n，netstat -rn
    • 🔺命令：uname -a，ps aux
    • 🔺命令：dpkg -l | head（rc–remove+配置文件保存）
/etc/passwd   -rw-r--r--
/etc/shadow   -rw-r-----
  • Windows
    • 🔺命令：ipconfig /all,ipconfig /displaydns（DNS缓存）,netstat -bnao,netstat -r（查询路由）
    • 🔺命令：net view,net view /domain（网络共享信息）
    • 🔺命令：net user /domain,net user %username% /domain（域环境的信息）
    • 🔺命令：net accounts,net share（机器开的共享）
    • 🔺命令：net localgroup administrator username /add（加入管理员组）
    • 🔺命令：net group "Domain Controllers" /domain（查看域控制器的主机账号）
    • 🔺命令：net share name$=C:\ /unlimited（域控制器的共享）
    • 🔺命令：net user username /active:yes /domain（域管理员启动锁定的账号）

3. WMIC（WINDOWS MANAGEMENT INSTRUMENTATION）

  • 🔺命令：wmic nicconfig get ipaddress,macaddress（读取IP地址，MAC地址）
  • 🔺命令：wmic computersystem get username（当前登陆用户账号）
  • 🔺命令：wmic netlogin get name,lastcalclogon（查看用户登陆记录，通过网络login）
  • 🔺命令：wmic process get caption,executablepath,commandline（查看当前进程信息，由哪些命令执行，执行程序的路径）
  • 🔺命令：wmic process where name="calc.exe" call terminate（结束一个进程）
  • 🔺命令：wmic os get name,servicepackmajorversion（提取操作系统版本，补丁）
  • 🔺命令：wmic product get name,version（查看当前操作系统安装的软件）
  • 🔺命令：wmic product where name="name" call uninstall /nointeractive（删除软件，静默的去删除）
  • 🔺命令：wmic share get /ALL（查看共享文件夹）
  • 🔺命令：wmic /node:"machinename" path Win32_TerminalServiceSetting where AllowTSConnections="0" call SetAllowTSConnections "1"（开启远程桌面） ▲ 2019.12.21 Win10的RPC服务器不可用，可在系统属性 → 远程中开启远程协助
  • 🔺命令：wmic nteventlog get path,filename,writeable（查看当前系统日志对应操作系统文件）
查看操作系统日志：运行 → eventvwr

5. 收集敏感信息

  • 商业信息
  • 系统信息
  • Linux
    • /etc；/usr/local/etc
    • /etc/passwd；/etc/shadow
    • .ssh；.gnupg 公私钥（默认隐藏文件）
    • The e-mail and data files
    • 业务数据库：身份认证服务数据库
    • /tmp
  • Windows
    • SAM数据库；注册表文件
    • %SYSTEMROOT%\repair\SAM（SAM数据库备份）
    • %SYSTEMROOT%\System32\config\RegBack\SAM（SAM数据库备份）
    • 业务数据库：身份认证数据库
    • 临时文件目录
      • UserProfile\AppData\Local\Microsoft\Windows\Temporary Internet Files\

6. 隐藏痕迹

  • Windows
    • 禁止在登陆界面显示新建账号
    • 🔺命令：REG ADD "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindosNT\CurrentVersion\WinLogon\SpecialAccounts\UserList" /v uname（要隐藏的账号名）/T REG_DWORD /D 0 ▲ 2019.12.21 Win10命令可以执行，但未禁止
    • del %WINDIR%*.log /a/s/q/f（强制安静不留痕迹删除）
  • Linux
    • 🔺命令：history（重点关注，查看曾经运行的命令）

history -c（擦除history之前的命令）
lsattr（查看权限相关文件的属性）

  • rm .bash_history && chattr +i .bash_history（文件名）（文件不能再写入）
history存在内存中，关机即删除
.bash_history存在硬盘，关机不删除
  • 日志（/var/log）
    • auth.log/secure
    • btmp/wtmp（数据文件，不能直接查看，通过🔺命令：lastb/last查看文件）
    • 🔺命令：lastlog/faillog
  • 其他日志和HIDS等
