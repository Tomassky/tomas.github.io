2. 本地提权

  • 系统账号之间权限隔离（系统账号和应用程序账号）

    • 服务账号无需用户登陆已在后台启动服务

4. 本地提权

  • Windows
    • User
    • Administrator
    • System（启动运行系统内核级别的权限，真正有最大权限的账号）

    • User
    • Root
