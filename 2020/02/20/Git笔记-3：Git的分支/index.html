1. Git的分支

  • 版本发布、功能增加、错误修改

  • 分支发布合并

2. Git分支操作

  • 创建分支：🔺命令：git branch 分支名
    • 基于某个分支创建的分支，会复制其内容
  • 查看分支：🔺命令：git branch
  • 跳转分支：🔺命令：git checkout 分支名
  • 创建分支并跳转：🔺命令：git checkout -b 分支名
  • 删除分支：🔺命令：git branch -d 分支名
    • ▲ 无合并到master的分支无法删除，或者使用-D参数`**
  • 合并分支：🔺命令：git merge 分支名
