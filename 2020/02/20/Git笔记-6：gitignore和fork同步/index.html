1. gitignore/fork同步

  • 创建一个.gitignore文件，保存要忽略的内容
    • 忽略系统生成的文件，如IDE的配置
    • 忽略编译生成的中间文件、可执行文件等
    • 忽略敏感的配置文件和本地不想提交的脚本
  • 常用参数
    • 配置文件：.settings/
    • 脚本文件：*.sh
    • 上次所有txt文件：!*.txt
    • 某个目录：/a/*.class
  • gitignore网站：https://www.gitignore.io
  • 查看远程的版本信息：🔺命令：git remote -v
  • 增加一个远程仓库上传流：🔺命令：git remote add upstream https://...
  • 更新远程仓库的版本：🔺命令：git fetch upstream 分支名
    • 分支名默认省略，则为master分支
    • ▲ pull = fetch + merge
  • 查看远程原始仓库的分支：🔺命令：git branch -r
