Kali Linux笔记(七十三)：ICMP隧道--ptunnel

发表于 2020-02-02 | 分类于 Kali

1. ptunnle

  • Ping tunnel ICMP隧道工具
    • 通过ICMP echo（ping requests，8）和reply（ping reply，0）实现隧道
    • 适用于防火墙只允许ping出站流量的环境
    • 支持多并发连接、性能优
    • 支持身份验证
    • 需要root权限
    • 支持抓包
      • Win：winpcap
      • Linux：libpcap
    • 工作过程（指定Proxy的时候，需要指定目的地，客户端指定）
      • Proxy、Client、Destination
  • 服务器
    • 🔺命令：ptunnel -x 1234
  • 客户端
    • 🔺命令：sudo ptunnel -p proxy -lp 2222 -da destination -dp 22 -x 1234
  • 嵌套SSH隧道
    • 🔺命令：ssh -CNfg -D 7000 root@127.0.0.1 -p 2222
  • ▲ ptunnel直到目前的最新版仍存在拒绝服务漏洞
    • 0.72

2. 实验环境准备

  • Kali（Proxy&Destinatiom）
    • 网卡方式为Bridge方式，桥接到无线网络，检查网络
  • Mono防火墙
    • 第一个网卡为Hosy-only方式，第二个网卡为Bridge桥接方式，桥接到无线网络，检查
  • Kali（内网）
    • 网卡方式为Host-only方式，检查网络
    • 浏览器访问mono，检查防火墙规则
      • Protocol → ICMP
      • Sourece → LAN subnet
      • LAN DNS -> WAM PING
    • ping/上网尝试
  • Window XP SP3
    • 网卡方式为Host-only方式，检查网络

3. 实验

  • Kali（Proxy&Destinatiom）
    • 服务器：🔺命令：ptunnel -x 1234
    • 启动Wireshark抓包，查看发送的数据包内容（ICMP数据包）
  • Kali（内网）
    • 🔺命令：sudo ptunnel -h
    • 客户端：🔺命令：sudo ptunnel -p Kali的IP -lp 2222 -da 192.168.0.1 -dp 80 -x 1234
    • 浏览器访问127.0.0.1:2222进行尝试
  • Kali（Proxy&Destinatiom）
    • 开启SSH（修改SSH配置文件）
  • Kali（内网）
    • 🔺命令：sudo ptunnel -p Kali的IP -lp 2222 -da Kali的IP -dp 22 -x 1234
    • 🔺命令：ssh -CNfg -D 7000 root@127.0.0.1 -p 2222
  • Window XP SP3
    • 设置浏览器Internet代理，用内网Kali的IP:7000，上网尝试
