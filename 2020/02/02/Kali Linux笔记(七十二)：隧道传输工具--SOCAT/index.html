Kali Linux笔记(七十二)：隧道传输工具--SOCAT

发表于 2020-02-02 | 分类于 Kali
字数统计: 763 字 | 阅读时长 ≈ 3 分钟

1. SOCAT

  • 被称为nc++（增强版的nc）
    • 双向数据流通道工具
  • 连接端口
    • 🔺命令：socat -（标准的输入输出） tcp:1.1.1.1:80（需要连接的目标）
  • 侦听端口
    • 🔺命令：socat - tcp4-listen:22 / socat - tcp-l:33
  • 接受文件
    • 🔺命令：socat tcp4-listen:333 open:2.txt,creat,append
  • 发送文件
    • 🔺命令：cat 1.txt | socat - tcp4:1.1.1.1:333
  • 远程shell–服务器端
    • 🔺命令：socat tcp-l:23 exec:sh,pty,stderr
  • 端口转发
    • 🔺命令：socat tcp4-listen:22,fork tcp4:1.1.1.1:22
  • 远程执行命令
    • 服务器：🔺命令：socat - udp-l:2001
    • 客户端：🔺命令："id" | socat - udp4-datagram:1.1.1.1:2001
  • UDP全端口任意内容发包
    • 🔺命令：for PORT in {1..65535}; do echo “aaaaa" | socat - UDP4-DATAGRAM: 1.1.1.1:$PORT; sleep .1; done
  • 二进制编辑器
    • 🔺命令：echo -e "\0\14\0\0\c" | socat -u - file:/usr/bin/ squid.exe,seek,seek=0x00074420
  • 简单的WEB服务
    • 🔺命令：socat -T 1 -d -d TCP-L:10081,reuseaddr,fork,crlf SYSTEM:"echo -e \"\\ \"HTTP/1.0 200 OK\\\nDocumentType: text/plain\\\n\\\ndate: \$\(date\)\\ \nserver:\$SOCAT_SOCKADDR:\$SOCAT_SOCKPORT\\\nclient: \ $SOCAT_PEERADDR:\$SOCAT_PEERPORT\\\n\\\"\"; cat; echo -e \"\\\"\\\n\ \\"\""

2. 实验环境准备

  • Kali
    • 网卡方式为Bridge方式，桥接到无线网络，检查网络
  • bodhi
    • 网卡方式为Bridge方式，桥接到无线网络，检查网络

3. TCP连接/侦听端口

  • Kali
    • 🔺命令：socat - tcp:192.168.0.1:80
    • 🔺命令：get / http 1.1/
    • 🔺命令：head
    • 侦听端口：🔺命令：socat - tcp4-listen:2222（作为服务器端使用）
  • bodhi
    • 🔺命令：apt-get install socat
    • 🔺命令：socat - tcp:Kali的IP:2222
    • 进行传输数据测试

4. TCP传送/接收文件

  • Kali
    • 🔺命令：socat tcp4-listen:333 open:2.txt,creat,append（接收文件）
  • bodhi
    • 🔺命令：echo "asdasdsafasfs" > 1.txt
    • 🔺命令：cat 1.txt | socat - tcp4:Kali的IP:333（发送文件）

5. TCP远程Shell–服务器端

  • Kali
    • 🔺命令：socat tcp-l:23 exec:sh（shell）,pty（终端）,stderr（标准输入输出）
  • bodhi
    • 🔺命令：socat - tcp:Kali的IP:23
    • 🔺命令：id/ifconfig进行测试

6. TCP端口转发

  • Kali
    • 🔺命令：socat tcp4-listen:8080,fork tcp4:192.168.0.1:80
  • bodhi
    • 浏览器访问Kali的IP:8080

7. UDP远程执行命令

  • Kali
    • 🔺命令：socat - udp-l:2000
  • bodhi
    • 🔺命令：echo "id"" | socat - udp4-datagram:Kali的IP:2000

8. UDP全端口任意内容发包（类似全端口扫描）

  • Kali
    • Wireshark抓包，过滤UDP数据包
  • bodhi
    • 🔺命令：for PORT in {1..65535}; do echo "aaaaa" | socat - UDP4-DATAGRAM: Kali的IP地址:$PORT; sleep .1; done
  • ▲ 2020.2.2 发送的数据包应用层为ADwin configuration protocol，也为运输层UDP包

9. 二进制编辑器

  • 🔺命令：echo -e（使用二进制） "\0\14\0\0\c" | socat -u（单向数据传输） -file:/usr/bin/ squid.exe,seek,seek=0x00074420（偏移量）
  • ▲ 2020.2.2 未实验，此段代码为一个翻墙软件在Window平台正常启动所需代码，需进行偏移

10. TCP简单的WEB服务器

  • Kali
    • 🔺命令：socat -T 1 -d -d TCP-L:10081,reuseaddr,fork,crlf SYSTEM:"echo -e \"\\ \"HTTP/1.0 200 OK\\\nDocumentType: text/plain\\\n\\\ndate: \$\(date\)\\ \nserver:\$SOCAT_SOCKADDR:\$SOCAT_SOCKPORT\\\nclient: \ $SOCAT_PEERADDR:\$SOCAT_PEERPORT\\\n\\\"\"; cat; echo -e \"\\\"\\\n\ \\"\"""
  • bodhi
    • 浏览器访问Kali的IP:10081
    • 字符界面：🔺命令：socat - tcp:Kali的IP:10081
