1. NCAT

  • 被称为众多NC衍生版软件中最优的选择
  • 代理功能（如果不指定端口，默认为31337）
    • 🔺命令：ncat -l 8080 --proxy-type http --proxy-auth user:pass
    • 浏览器设置代理服务器127.0.0.1:31337，进行网页访问
  • Broke中介功能
    • AB不同但AC、BC互通
    • 服务器：🔺命令：ncat -l 333 --broke
    • 客户端之间：发送任何信息都会被hub到其他客户端
    • 批量执行命令：🔺命令：ncat 1.1.1.1 --sh-exec "echopwd"
    • 批量传文件：🔺命令：ncat --send-only 1.1.1.1 < inputfile
  • 实验（Kail上的三个终端窗口进行模拟）
    • 1：🔺命令：ncat -l 333 --broke
    • 2/3：🔺命令：ncat localhost 333
    • 2：🔺命令：aaa → 3收得到信息
    • 3：🔺命令：bbb → 2收得到信息
    • 2：🔺命令：ncat 127.0.0.1 333 --sh-exec "echopwd" → 3进行pwd系统指令操作
    • 3：🔺命令：ncat --send-only 127.0.0.1 333 < a.php → 2收到a.php文件
