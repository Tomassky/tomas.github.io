1. 文件操作

  • fopen() 的第一个参数包含被打开的文件名，第二个参数规定打开文件的模式。
    • $myfile = fopen(“webdictionary.txt”, “r”) or die(“Unable to open file!”);
  • 读文件
    • Fgets($find) //读第一行文件
      • ▲ 文件指针必须是有效的，必须指向由 fopen()或fsockopen()成功打开的文件(并还未由 fclose() 关闭)。
      • ▲ fget()可以被用来模拟get/post请求，结合fopen()和fsockopen()两个函数
    • Fread($find,获取字节) //指定获取内容
    • Filesize($find) //获取文件字节
      • ▲ $find 必须为路径
    • Fclose($find) //关闭资源
  • 修改或添加文件内容
    • Fopen(./1.xx,”a+”);
    • Fwrite(“文件”,”内容”);
      • ▲ 只能将字符串string写进文件，无法将数组写进
      • ▲ 写进数组可借用file_put_contents()函数
    • Fclose(find); //关闭资源
  • 文件指针feof()
    • ▲ 文件指针必须是有效的，必须指向由 fopen()或fsockopen()成功打开的文件（并还未由 fclose()关闭）
<?php

    //读文件
    $f = fopen("./1.txt", "a+");

    echo Fgets($f);  //读第一行文件
    echo Fgets($f);  //读第二行文件
    echo Fread($f,2);  //获取文件前两个字符

    echo Filesize($f);  //获取屋内按总的字符数
    echo Fread($f,Filesize('./1.html'));  //输出1.html中的所有内容

    Fclose($f);  //关闭文件
    echo Fread($f,2);  //无法读取到文件

    //读写文件
    $f = fopen("./1.txt", "a+");

    Fwrite($f,'aaaaaa')  //在文件最后添加aaaaaa
    Fclose($f);

    $f = fopen("./1.txt", "w");

    Fwrite($f,'aaaaaa')  //删除文件，并在文件开头添加aaaaaa
    Fclose($f);

    //路径拼接

    $lujing = $_SERVER['DOCUMENT_ROOT']."/";

    $f = fopen($lujing."123.txt","a.txt");

    Fwrite($f,'aaaaaa\r\n');  //写入成功

    //命令读取文件

    echo `dir`;  //读取系统路径

    echo `type d:\\1.txt`;  //读取1.txt文件中的内容

    //文件指针

    $f = fopen("./1.txt", "r+");
    while(!feof($f))
    {
        $result.= fgets($f,128);
    }
    echo $result;  //输出1.txt的所有内容
?>
