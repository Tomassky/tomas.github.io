1. 分支结构

  • 在PHP中，分支结构主要有两种：if分支和switch分支

2. if分支

  • 如果的意思，给定一个条件，同时为该条件设置多种（两种）情况，然后通过条件判断来实现具体的执行段
  • ▲ 如果条件特别多才会采用复合if形式
  • ▲ 如果条件比较单一（同一个条件），会采用elseif复合方式
  • ▲ 如果判断条件不一致，建议使用嵌套语法（不宜有太多层嵌套：影响代码美观）
<?php


    //分支结构：if分支


    //最简if
    $day = 'weekenday';

    //如果是星期天就出去玩
    if($day == 'sunday'){
        echo 'go out';
    }


    //基本if判断
    //如果是星期天就出去玩，否则上班
    if($day == 'sunday'){
        echo 'go out play';
    }else{
        //不满足条件
        echo 'work';
    }


    //复合if结构
    //如果是星期天就出去玩，否则不能，但是如果是周六，那么可以在家玩
    echo '<hr/>';
    $day = 'satday';
    if($day == 'sunday'){
        echo 'go out';
    }else{
        //包含全部不满足情况
        //重新进行判断
        if($day == 'satday'){
            echo 'play at home';
        }else{
            echo 'work';
        }
    }

    echo '<hr/>';
    if($day == 'sunday'){
        echo 'go out';
    }elseif($day == 'satday'){
        echo 'play at home';
    }else{
        echo 'work';
    }

?>

3. switch分支

  • 有一组情形存在，同过一条件，通常有多个值，但是每一个值都会有对应不同的代码要执行
  • ▲ 在switch中，如果条件匹配成功，那么系统就不会再次匹配条件，会自动顺序执行向下的所有代码（case代码除外），需要中断执行：break表示中断switch（结束）
<?php

    //分支结构：switch分支

    //根据日期做不同的事情
    $day = 1;

    //从1到5做不同的事情
    switch($day){
        //$day条件一定是个具体的值
        case 1: //$day == 1
            echo '1';
            //break;
        case 2:
            echo '2';
            break;
        case 3:
            echo '3';
            break;
        case 4:
            echo '4';
            break;
        case 5:
            echo '5';
            break;
        default:
            echo 'error';
            break;  
    }

?>
