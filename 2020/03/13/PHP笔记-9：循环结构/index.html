1. 循环结构

  • 代码段在一定的控制下，可以多次执行
  • 循环结构
    • For循环：通过条件、起始和终止判断执行
    • While循环：通过判断条件终止
    • Do-while循环：跟while差不多
    • Foreach循环：专门针对数组

2. for循环

<?php

    //循环结构：for循环


    //从1到10输出：初始为1，截止为10
    for($i = 1; $i <= 10;$i++){
        //输出
        //echo $i,'<br/>';
    }

    //最后：$i == 11
    echo $i;  //输出为 11
?>

3. while循环

<?php

    //while循环


    //定义条件
    $i = 1;

    //循环判定执行
    while($i <= 10){
        //循环体
        echo $i++,'<br/>';

        //循环条件变更
        //$i++;
    }
?>

4. do-while循环

  • 看着很像while，while首先进行条件判定然后执行循环体，有可能出现第一次就条件不满足，那么就会直接失败（循环体一次都不执行）。Do-while就是先干了再说（执行循环体），后判断条件。（至少会执行一次循环体）
<?php

    //do-while循环结构

    //输出1-10之间的偶数（条件判定加入）

    //定义基础条件
    $i = 1;

    //循环判定
    do{
        //执行输出
        if($i % 2 != 1){
            //是偶数
            echo $i,'<br/>';        
        }

        //条件变更
        $i++;
    }while($i <= 10);
?>

5. 循环控制

  • 循环控制：在循环内部对循环本身进行控制
  • 中断控制：重新开始循环，循环体中还有其他内容，也再执行
    • Continue 层级; //默认是1（循环可以多层嵌套）
  • 终止控制：循环直接结束
    • Break 层级; //默认是1
  • 嵌套循环控制
    • Continue 2; //当前自己循环后面内容不再执行，同时外部循环如果还有循环体也不再执行，重新来过
    • Break 2; //当前自己循环结束，同时外部也结束（如果还有外部不受影响，继续执行）
//字符串函数只能给字符串使用
<?php

    //循环控制


    //需求：输出1到100之间的5的倍数

    $i = 1;

    while($i <= 100){

        //判断：是否是5的倍数
        if($i % 5 != 0) {
            //说明当前$i不是5的倍数

            //重新循环
            $i++;

            //重新循环
            continue;         //系统重新跳到循环开始处


            //终止循环
            //break;
        }


        //输出数值
        echo $i++,'<br/>';
    }

    echo $i;

    //嵌套的循环控制
    $i = 1;
    $j = 1;

    for($i=1;$i<=10;$i++)
    {
        for($j=1;$j<=2;$j++)
        {
            echo 'ok!';
            //break;  //输出10个ok!
            break 2;  //输出1个ok!
        }
    }

?>

6. foreach循环

<?php
    $arr1 = array('mingzi' => '心东','nianling' => '21','zhiye' => '网络安全','aihao' => '吃饭睡觉');

    foreach($arr1 as $key => $value)
    {
        echo '键名:'.$key.'值'.$value.'</br>';
        echo $arr1[$key]."</br>";  //输出关联数组
    }
?>

7. 流程控制替代语法

  • 替换模式
    • 左大括号{使用冒号替代：
    • 右大括号}使用end+对应的起始标记替代
  • 具体替换模式
    • if(): endif;
    • sswitch(): endswitch;
    • for(;;): endfor;
    • while(): endwhile;
    • foreach(): endforeach;
  • 需求：打印一个九九乘法表，使用表格来展示
<table border=1>
    <?php for($i = 1;$i < 10;$i++){?>
        <tr>
            <?php for($j = 1;$j <= $i;$j++){?>
                <td>
                    <?php echo $i . ' * ' . $j . ' = ' . $i * $j;?>
                </td>
            <?php }?>
        </tr>
    <?php }?>
</table>


<table border=1>
    <?php for($i = 1;$i < 10;$i++):?>
        <tr>
            <?php for($j = 1;$j <= $i;$j++):?>
                <td>
                    <?php echo $i . ' * ' . $j . ' = ' . $i * $j;?>
                </td>
            <?php endfor;?>
        </tr>
    <?php endfor;?>
</table>
