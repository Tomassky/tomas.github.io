1. MySQL命令符

  • Mysqladmin 日常管理命令总结 或 重置密码
  • Mysql 用到操作数据库
  • Mysqld 用来开启MYSQL 或者 关闭
  • Mysqlimport 数据库导入

2. MySQL操作

  • 登录MYSQL mysql -h 192.168.1.100 -u root -p -P 123
    • -h 你的IP
    • -U 你的用户名
    • -p你的密码 可不填 但是后面要填
    • -P （大写） 端口号
  • 常规操作
    • 查看所有数据库：Show databases
    • 选择数据库之后 查看当前数据库所有的表：show tables;
    • 创建数据库：Create database 数据库名;
    • 选择数据库：use 数据库名;
    • 创建表跟字段
      • Create table admin2(
      • Username varchar(20) not null,
      • Passworld text not null);
    • 查看表结构：desc 表名
    • 添加数据
      • Insert into 表名 values (“”);
      • Insert into 表名 (字段名) values (“”);
    • 查询数据库
      • Select * from 表名;
    • 修改数据
      • Update 表名 set 字段 = ‘’;
    • 清空表数据
      • Delete from 表名
    • 删除表
      • Drop table 表名;
    • 备份数据库
      • 命令行备份
        • select * from 表名 into outfile ‘/tmp/1.txt’
        • ▲ 可能报错，需要修改–secure-file-priv的路径，可绕过–通过日志绕过
      • 界面备份
        • 从mysql的数据库路径/data，拷走所有的表结构以及表数据
    • 自增型
      • auto_increment
    • 搜索关键字
      • select * from admin where username like “%%xindong”;
    • 分页

3. PHP操作MySQL数据库

<?php

    $lin = mysqli_connect('127.0.0.1','root','root');

    //数据库使用--mysqli_query

    mysqli_query($lin,"use php_mysql_demo");  //选择数据库
    mysqli_query($lin,"set names utf8");  //设置数据库编码

    mysqli_query($lin,"insert into admin values ('xindong','123')");  //添加数据库记录
    mysqli_query($lin,"delete from admin where username = '1111'");  //删除数据库记录
    mysqli_query($lin,"update admin set username = '1111'");  //修改数据

    //数据库查询使用--mysqli_fetch_array
    /*
    mysqli_fetch_array()包含两个内容，默认的，函数可以通过第二个参数来设定返回的类型
    1. MYSQLI_ASSOC：返回关联数组
    2. MYSQLI_NUM：返回索引数组
    3. MYSQLI_BOTH：默认的，两种数值都返回
    */

    $list = mysqli_query($lin,"select * from admin");
    $list_data = mysqli_fetch_array($list,MYSQLI_ASSOC);
    var_dump($list_data);  //打印单行数据，指针默认从顶部开始
    echo "</br>";


    //输出查询的整个数据库
    $list1 = mysqli_query($lin,"select * from admin");
    $arr = array();
    while($list1_data = mysqli_fetch_array($list1,MYSQLI_ASSOC))
    {
        $arr[] = $list1_data;
        //直接输出 var_dump($list1_data);
    }
    var_dump($arr);  //打印整个数据
