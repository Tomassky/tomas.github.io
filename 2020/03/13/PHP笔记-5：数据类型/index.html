1. 数据类型

  • ▲ 数据类型：data type，在PHP中指的是存储的数据本身的类型，而不是变量的类型。PHP是一种弱类型语言，变量本身没有数据类型。
  • 简单（基本）数据类型：4个小类
    • 整型：int/integer，系统分配4个字节存储，表示整数类型（有前提）
    • 浮点型：float/double，系统分配8个字节存储，表示小数或者整型存不下的整数
    • 字符串型：string，系统根据实际长度分配，表示字符串（引号）
    • 布尔类型：bool/boolean，表示布尔类型，只有两个值：true和false
  • 复合数据类型：2个小类
    • 对象类型：object，存放对象（面向对象）
    • 数组类型：array，存储多个数据（一次性）
  • 特殊数据类型：2个小类
    • 资源类型：resource，存放资源数据（PHP外部数据，如数据库、文件）
    • 空类型：NULL，只有一个值就是NULL（不能运算）
  • 类型转换
    • 自动转换：系统根据需求自己判定，自己转换（用的比较多，效率偏低）
    • 强制（手动）转换：认为根据需要的目标类型转换
      • 强制转换规则：在变量之前增加一个括号()，然后在里面写上对应类型：int/integer….其中NULL类型用到unset()
    • 其他类型转数值的说明
      • 布尔true为1，false为0
      • 字符串转数值有自己的规则
        • 以字母开头的字符串，永远为0
        • 以数字开头的字符串，取到碰到字符串为止（不会同时包含两个小数点）
<?php

    //数据类型

    $a = 'abc1.1.1';
    $b = '1.1.1abc';

    //自动转换
    echo $a + $b;  //输出为 1.1  $a -> 0,$b -> 1.1

    //强制转换
    echo '<br/>',(float)$a,(float)$b;  //输出为 01.1  $a -> 0,$b -> 1.1

?>
  • 类型判断
    • 通过一组类型判断函数，来判断变量，最终返回这个变量所保存数据的数据类型（相同结果为true，失败为false）：是一组以is_开头后面跟类型名字的函数：is_XXX(变量名)
    • Bool类型不能用echo来查看，可以使用var_dump结构查看
    • Var_dump(变量1,变量2…)
    • Gettype(变量名)：获取类型，得到的是该类型对应的字符串
    • Settype(变量名,类型)：设定数据类型 -> 与强制转换不同
      • 强制转换(类型)变量名，是对数据值复制的内容进行处理（不会处理实际存储的内容）
      • settype会直接改变数据本身
<?php

    //判断数据类型
    echo '<hr/>';
    var_dump(is_int($a));       //输出为 bool(false)
    var_dump(is_string($a));      //输出为 bool(true)

    echo '<hr/>';
    echo gettype($a),'<br/>';  //输出为 string

    //设置类型
    var_dump(settype($b,'int'));  //输出为 bool(true)
    echo gettype($b),$b;   //输出为 integet1

?>

2. 整型

  • 保存整数数值（范围限制），4个字节存储数据，最大就是32位：42亿多。但是在PHP中默认是有符号类型（区分正负数）
  • 进制转换函数
    • Decbin()：十进制转二进制
    • Decoct()：十进制转八进制
    • Dechex()：十进制转十六进制
    • Bindec()：二进制转十进制
<?php

    //简单数据类型：整型、浮点型和布尔型

    //定义4种整型数据

    $a1 = 110;
    $a2 = 0b110;
    $a3 = 0110;
    $a4 = 0x110;

    echo $a1,'~',$a2,'~',$a3,'~',$a4,'<hr/>'; //输出为110~6~72~272,默认的PHP输出数值都会自动转换成10进制输出

    /*
    十进制转二进制
    10 ---》8 + 2 ---》2^3 + 2^1 ---》从二进制右侧开始，按照对应的指数次位置补1，没有的补0


    从最后侧开始：00000000 00000000 00000000 00001010  


    二进制转十进制
    01101011 ==>从右侧开始 1*2^0 + 1*2^1 + 0*2^2 + 1*2^3 + 0*2^4 + 1*2^5 + 1*2^6 + 0*2^7
    == 1 + 2 + 0 + 8 + 0 + 32 + 64 + 0 == 107

    */

    //利用进制函数运算
    var_dump(decbin(107));  //十进制转二进制,输出为 string(7) "1101011"
    echo "</br>";
    var_dump(dechex(107));  //十进制转十六进制,输出为 string(2) "6b" 
    echo "</br>";
    var_dump(decoct(107));  //十进制转八进制,输出为 string(3) "153" 
    echo "</br>";
    var_dump(bindec(1101011));  //二进制转十进制,输出为 int(107)
    echo "</br>";

?>

3. 浮点型

  • 小数以及超过整型所能存储范围的整数（不保证精度），精度范围大概在15个有效数字左右
  • 浮点型定义有两种方式
    • $f = 1.23;
    • $f = 1.23e10; //科学计数法，其中e表示底10
  • ▲ 尽量不用用浮点数做精确判断：浮点数保存的数据不够精确，而且在计算机中凡是小数基本上存的都不准确
<?php

    //浮点数
    $f1 = 1.23;
    $f2 = 1.23e10; 
    $f3 = PHP_INT_MAX + 1;    //整型超过自身存储的大小之后会用浮点型存储

    var_dump($f1,$f2,$f3);  //输出为 float(1.23) float(12300000000) float(9.2233720368548E+18) 
    echo "</br>";

    //浮点数判断
    $f4 = 0.7;
    $f5 = 2.1;
    $f6 = $f5 / 3;

    echo $f6;  //输出为 0.7
    var_dump($f4 == $f6);  //输出为 bool(false)

    echo '<hr/>';

?>

4. 布尔型

  • 两个值true和false，通常是用于判断比较
  • 在进行某些数据判断的时候，需要特别注意类型转换
    • Empty()：判断数据的值是否为”空”，不是NULL，如果为空返回true，不为空返回false
    • Isset()：判断数据存储的变量本身是否存在，存在变量返回true，不存在返回false
      • ▲ 不能直接传输参数，只能传入变量
<?php

    /布尔类型
    $b1 = true;
    $b2 = FALSE;

    var_dump(empty(0));  //输出为 bool(true) 
    echo "</br>";

    //var_dump(isset(0));  //报错
    var_dump(isset($b2));  //输出为 bool(true) 
    echo "</br>";

    var_dump($b1,$b2);  //输出为 bool(true) bool(false) 
    echo "</br>";

?>
